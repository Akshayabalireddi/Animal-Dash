                         HOW TO RUN (3 STEPS)
================================================================================

STEP 1: Open this folder in VSCode
        File → Open Folder → Select the "Level-4" folder

STEP 2: Install pygame (if you don't have it)
        Open terminal (Ctrl+`) and type:
        pip install pygame

STEP 3: Run the game
        Option A: Press Ctrl+F5 (Run Python File)
        Option B: Open terminal and type: python main.py

        ✅ GAME LAUNCHES!

================================================================================
                          TROUBLESHOOTING
================================================================================

Problem: "No module named pygame"
Solution: pip install pygame

Problem: Game won't launch
Solution: Make sure you have Python 3.7+
         Check that main.py is in this folder

Problem: Assets showing as magenta boxes
Solution: Create "assets/sprites/lion/" folder with sprite files
         Game still works! Boxes are just placeholders

Problem: Game runs too fast/slow
Solution: Check FPS setting in main.py (line ~14)

Problem: Hunter is too easy/hard
Solution: Edit Hunter parameters in main.py (search "Hunter" class)
